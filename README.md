# 🧑‍💻 Registro de Ponto por Reconhecimento Facial

## 🚀 Projeto: Registro de Ponto

**Tags:** Django • Python

Desenvolvimento de um sistema de registro de ponto em **Python com Django e OpenCV**, com integração de **reconhecimento facial em tempo real**.

O fluxo inicia com registro manual e evolui para uma verificação automatizada, proporcionando **mais eficiência, segurança e precisão** no controle de presença.

---

## 📒 README.md

## 📝 Descrição do Projeto

O **Registro de Ponto com Reconhecimento Facial** é um sistema completo que permite o controle de presença utilizando **identificação facial automatizada**.

A solução inclui:

* 🔗 **API REST** para integração com outros sistemas
* 📱 **Aplicativo mobile (Kivy)** para uso em dispositivos móveis
* 🎯 **Reconhecimento facial em tempo real** para validação de identidade

---

## ⚙️ Funcionalidades

### 👤 Registro de Ponto por Reconhecimento Facial

* ✍️ **Funcionários:**
  Gerenciamento completo de colaboradores, incluindo:

  * Foto
  * Nome
  * Documento
  * Captura de faces para treinamento do modelo

* 📷 **Reconhecimento Facial:**
  Identificação automática do usuário para registro de ponto

* 🧾 **Histórico de Registros:**
  Consulta de registros por data e hora

---

### 📲 Integração com Aplicativo Mobile (Kivy)

* Aplicativo desenvolvido em **Kivy/KivyMD**
* Comunicação direta com a API Django
* Registro de ponto via dispositivo móvel

---

## 🔗 API REST

* 📡 **Endpoints de Reconhecimento e Cadastro:**
  Permitem o consumo e envio de dados relacionados ao reconhecimento facial

* 🔐 **Operações Seguras:**
  Registro e consulta de pontos via API

## 📋 Pré-requisitos

Antes de executar o projeto, certifique-se de possuir:

* Python 3.9
* Django (Django==4.2)
* Django REST Framework (djangorestframework==3.15.2)
* Requests (requests==2.32.3)
* OpenCV

  * opencv-python==4.5.5.64
  * opencv-contrib-python==4.5.5.64
* Numpy (numpy==1.24.4)
* Kivy (para desenvolvimento mobile)
